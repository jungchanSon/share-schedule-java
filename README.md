### git branch

```text
feature/share-{yyyyMMddHHmmss}      ### 기능 관련 (api 개발, 수정 등)
infra/share-{yyyyMMddHHmmss}        ### 인프라 관련 (docker, sql 등)
bugfix/share-{yyyyMMddHHmmss}       ### 버그 픽스 관련 (버그 수정)

ex) feature/share-20240629112315
```
